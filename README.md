# Paddle-DDSP

## Usage
* Command version:

    ```bash
    $ python predict.py \
        [-c ckpt](optional) \
        [-i input audio (*.wav)](optional) \
        [-o output audio (*.wav)](optional)
    ```

* GUI version:

    ```bash
    $ python predict_gui.py
    ```
